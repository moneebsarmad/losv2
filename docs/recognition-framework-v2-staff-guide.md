# Staff Guide: Recognition Framework v2

> Staff select the behaviour they saw and the domain where it occurred. The system assigns the points. The same behaviour has the same value in every domain.

## Giving recognition

1. Search for and select the student. Select several students only when they performed the same positive action in the same event.
2. Select Righteousness, Responsibility, or Respect.
3. Select the one behaviour that best describes the main action.
4. Select where it occurred:
   - Prayer Space (Muṣallā)
   - Hallways & Transitions
   - Classroom & Learning
   - Lunch / Recess
   - Bathrooms
5. Review the fixed points and House shown by the system.
6. Add a note when the form requests one, then select **Award Points**.

One event receives one award. Recognise a positive action, choice, effort, skill, repair, or contribution—not merely the absence of misconduct.

## Notes

- `+5` and `+10` recognitions are designed to be quick and do not require a note.
- A `+20` recognition requires a short note naming what the student did and the meaningful challenge, pressure, conflict, harm, or difficulty involved.
- Students practising a replacement skill through a growth plan may be recognised more often, but the selected behaviour always keeps its fixed value.

## Exceptional `+50` recognition

Exceptional behaviours show a meaningful personal or social risk. Select the exceptional behaviour, enter a clear explanation, add witness information when helpful, and select **Submit Nomination**.

The nomination does not add points immediately. An authorised administrator reviews it. If approved, the system adds one `+50` award to the student and House totals. Exceptional nominations are submitted for one student at a time.

## Recognition narration

The review screen offers optional wording you can adapt when speaking with the student:

> “You showed [3R] by [behaviour]. That earned [points] points for your House.”

The purpose is specific feedback: name what the student did, connect it to the 3R, and make the recognition understandable.
